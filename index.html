Overview:

VisualHDD is a tool that scans the contents of your partitions, and generates a visual representation of those contents. This is useful for determining which files are taking up the most space on your drive. Furthermore, the program allows you to delete files from the representation directly, and quickly reconstructs the representation so that you don't even have to close the program to remove unnecessary files.

Notes: Work in progress

Feature List:

  • Multiple representations, choose the one that you like most
  • Optimized for efficiency
  • Delete files from within the program
